# セキュリティに関する注意事項

## 機密情報の管理

このリポジトリには**機密情報は一切含まれていません**。

### 除外されているファイル

以下のファイルは`.gitignore`により、Gitリポジトリから除外されています:

1. **`.env.local`** - 環境変数ファイル
   - API URLなどの設定を含む
   - ローカル開発環境でのみ使用
   - **絶対にコミットしないでください**

2. **`node_modules/`** - NPM依存パッケージ
   - 自動生成されるディレクトリ
   - `npm install`で再作成可能

3. **`.next/`** - Next.jsビルドファイル
   - ビルド時に自動生成
   - コミット不要

### 安全な使用方法

1. **初回セットアップ時:**
   ```bash
   # .env.exampleをコピー
   cp .env.example .env.local

   # .env.localを編集（機密情報は含まない）
   NEXT_PUBLIC_API_URL=http://localhost:8000
   ```

2. **本番環境:**
   - 本番環境の環境変数は、デプロイ先のプラットフォーム（Vercel、Netlifyなど）で設定
   - `.env.local`は本番環境では使用しない

### Google OAuth設定

Google OAuth Client IDは公開情報ですが、以下の理由により`.env.local`で管理します:

- 環境ごとに異なるClient IDを使用できる
- 設定の一元管理
- セキュリティベストプラクティスに従う

**重要:** Google OAuth Client **Secret**は、このフロントエンドアプリでは使用しません。バックエンドでのみ使用します。
